# normalization_rules.md
# ラグビーリーグワン パートナー企業名正規化ルール集
# Chat・Claude Code・Cowork の共通参照ファイル
# 更新日: 2026-03-19

---

## 1. 基本方針

B列の企業名を正式な法人名に変換してC列に記入する。
変換できない・すべきでない場合はB列をそのままC列にコピーするか、N列に理由を記入する。

---

## 2. STEP1: 略式法人格の変換（機械的・自動）

| 略式表記 | 変換後 |
|---|---|
| （株）/ (株) | 株式会社 |
| （有）/ (有) | 有限会社 |
| （合）/ (合) | 合同会社 |
| （社）/ (社) | 一般社団法人 |
| （財）/ (財) | 一般財団法人 |
| （公社）/ (公社) | 公益社団法人 |
| （公財）/ (公財) | 公益財団法人 |
| （医）/ (医) | 医療法人 |
| （協）/ (協) | 協同組合 |
| 株式會社 | 株式会社 |
| 有限會社 | 有限会社 |
| 會（旧字体） | 会 |

## 3. STEP2: 書式ルール（機械的・自動）

- **法人格直後のスペース削除**: 「株式会社 〇〇」→「株式会社〇〇」
- **全角英数字→半角**: Ａ→A、１→1 など（Unicode FF01-FF5E範囲・NFKC正規化）
- **全角スペース U+3000 → 半角スペース U+0020**: NFKC正規化の副作用として自動変換
- **全角ハイフン－(U+FF0D) → 半角ハイフン-(U+002D)**: NFKC正規化で自動変換
  - 例: 「Ｃ－ｐｏｓｉｔｉｏｎ株式会社」→「C-position株式会社」
  - **例外（保留して要確認）**: 全角ハイフン－の前後がカタカナ文字（U+30A0-U+30FF）の場合、長音符「ー」(U+30FC) の誤入力の可能性があるため自動変換せずユーザー確認する
    - 例: 「ヘネシ－」（誤: 全角ハイフン）→ 自動的に「ヘネシ-」にせず保留 → 確認後「ヘネシー」（長音符）に修正
- **その他の記号（〜・※・…・♪・＆ 等）**: 機械的な一括変換ルールは設けず、出現時に都度判断する。判断結果は本ルール（§3）に追記して再利用する。
- **辞書変換**: step2_dict.py の完全一致エントリに従い変換
- **括弧内の部署名・支店名・放送局コール等を削除**: 法人名に続く括弧内の付帯情報は除去する
  例: 「TOPPANエッジ株式会社(西日本営業統括本部 中四国営業本部)」→「TOPPANエッジ株式会社」
  例: 「山口放送株式会社(KRY)」→「山口放送株式会社」

## 4. 変換ルール

### 4-1. 略称・通称 → 正式法人名
辞書（step2_dict.py）に登録済みの略称はSTEP2で自動変換される。
辞書未登録のものはSTEP3でChat・Coworkが対応する。

### 4-2. グループ名義
持株会社または中核法人の正式名称に変換する。
例: 「〇〇グループ」→「〇〇ホールディングス株式会社」

### 4-3. 海外メーカーの日本法人（〇〇 ジャパン形式）
ブランド名と「ジャパン」の間のスペースを**保持**する。
例: 「アディダスジャパン株式会社」→「アディダス ジャパン株式会社」

### 4-4. 2社以上の併記
「株式会社〇〇／△△株式会社」形式で記載する。

### 4-5. 法人消滅・吸収合併済み
現存法人名を記載し、N列に合併情報を追記する。

### 4-6. 公営競技（競輪・競馬・競艇）
B列の表記をそのままC列にコピーする。
※ 公営競技はステータス除外の対象外（セクション6参照）。営業対象として残す。

### 4-7. 店舗名・施設名付きの法人名
法人名に続いて店舗名・施設名が記載されている場合は、本社法人名のみに変換する。
例: 「株式会社叶八 肉のたかお甘木店」→「株式会社叶八」
例: 「株式会社イズミゆめマートうきは」→「株式会社イズミ」

## 5. B列そのまま転記（変換しない）ケース

以下に該当する場合はB列の表記をそのままC列にコピーする：

- すでに正式な法人格付きの名称が記載されている
- **ホテル**: 施設名・支店名のまま転記（運営法人への変換不要）
  - 例: 「ホテルプラザ神戸」「ホテルマークワン」「三井ガーデンホテル柏の葉」
- **百貨店・支店名**: 支店名のまま転記（法人名への変換不要）
  - 例: 「高島屋柏店」「〇〇百貨店△△店」
- **商業施設・ショッピングモール**: 施設名のまま転記
  - 例: 「セブンパークアリオ柏」「玉川高島屋S.C.」「二子玉川ライズ」
- **駅ビル・駅チカ商業施設**: 施設名のまま転記

## 6. 除外ケース（N列に「除外」と記入）

以下に該当する場合はC列への記入は行わず、N列に「除外」と記入する：

### 業種による除外
- 病院・医院・クリニック・歯科・整形外科・治療院
- 介護施設・訪問看護・デイサービス・福祉施設
- 学校・大学・専門学校・幼稚園・保育園
- 士業（弁護士・税理士・社労士・行政書士等）
- 飲食店・食堂・カフェ・居酒屋・精肉店等の個人商店

### 除外対象外（業種に該当しても除外しないもの）
- **公営競技（競馬場・競輪場・競艇場・オートレース場）**: 施設運営は地方自治体・公営企業であり営業対象となり得るため除外しない
  - 例: 「JRA札幌競馬場」「平塚競輪場」「ホッカイドウ競馬」→ 除外しない

### 法人形態・実体による除外
- 有限会社で名称から小規模個人事業主が疑われるもの
- 屋号のみ・法人格なし・実体が確認できないもの
- 英字略称のみで正体不明のもの
- 画像ファイル名・ロゴファイル名（誤入力）
- 商店街振興組合・自治会・任意団体・同好会
- 部活動・スポーツスクール・ラグビースクール等の非営利団体
- NPO・一般社団法人のうち小規模地域団体
- 小規模地域協同組合（全国規模の農協・生協は対象）
- 市区町村レベルの観光協会・観光推進団体（都道府県・広域・全国規模は除外しない）

---

## 7. 要確認ケース（N列に「要確認」と記入）

以下に該当する場合はN列に「要確認」と記入し、C列は空欄のまま保留する：

- 同名複数法人が存在し、特定できないもの
- 公式サイト・公開情報から法人名が確認できないもの
- 法人の存続・解散が不明なもの
- 中国語・韓国語等の外国語表記で日本法人が特定できないもの
- グループ通称名・ホールディングス名義で登記法人名でないもの（例: げんき屋ホールディングス）は要確認とし、スポンサー契約の主体法人を特定する

## 9. 全国規模の協同組合・共済団体の支部名

全国農業協同組合連合会・全国労働者共済生活協同組合連合会（こくみん共済 coop）・全国生活協同組合連合会等、**全国規模の協同組合・共済団体は支部名・統括本部名を含む表記のままB列転記する。**
法人名のみに短縮したり、支部名・括弧書きを除去しない。

例:
- 「全国労働者共済生活協同組合連合会 九州統括本部(こくみん共済 coop)」→ そのままC列に転記

---

## 10. 短い・ユニークでない社名の扱い（要URL確認）

B列の名称が**短い・一般的なワード・英略語のみ**で構成されており、同名の別法人が複数存在する可能性がある場合は、辞書や知識だけで変換せず、**必ずH列のURLまたはWeb検索で法人を特定してからC列に書き込む。**

Coworkへ調査依頼する際は「H列のURLも確認した上で正式名称を確定してください」と明記する。

該当する目安:
- 4文字以下の英略称（例: Qsol、CTC）
- 日本語でも業種・地名と組み合わせただけの短い名称（例: フジタ、東和、大榮）
- 英語の一般名詞・社名（例: Marubeni、Kong Inc.）
- 有名企業の英字表記のみ（例: SECOM）は法人格欠落として正式社名に変換する（知識変換扱い）

---

## 11. 代理店経由ブランドの取り扱い

ブランド名の日本展開を代理店が担っている場合、代理店法人名ではなく**ブランド名のままB列転記する。**
代理店契約は変更される可能性があるため、代理店名への変換は行わない。

該当するケース:
- アンダーアーマー（日本展開: 株式会社ドーム）→「アンダーアーマー」のままC列に転記
- 同様のブランド代理店契約が確認された場合も同じ扱いとする

なお、ブランドに日本法人が存在する場合（例: アディダス ジャパン株式会社）は
日本法人の正式名称に変換する（§4-3参照）。

### 日本ブランド名・メーカー名と運営会社名が異なる場合
スポンサー名が日本のブランド名・商品名であり、運営会社名と異なる場合は
**運営会社の正式法人名に変換する**。

該当するケース:
- スポンサー表記がブランド名のみで法人格がない（例: エルキュール → 株式会社mood Inc.）
- 公式サイトの会社概要・法人番号検索で運営会社を特定してから変換する

§11の他のケースとの使い分け:
- 海外ブランドの日本代理店 → ブランド名のまま転記（代理店名に変換しない）
- 海外本社の日本法人（〇〇 ジャパン形式）→ 日本法人の正式名称に変換
- 日本ブランド名≠日本運営会社名 → 運営会社の正式法人名に変換（本項）

---

## 12. 変換確認優先順位（Cowork・調査時）

1. H列の公式サイトURLを確認
2. 公式サイトの会社概要・特商法表記ページを確認
3. gBizINFO・国税庁法人番号公表サイトで法人番号を確認
4. Web検索で公式プレスリリース・ニュース記事を確認
5. 同名複数法人がある場合は所在地（都道府県・市区町村）で絞り込む
